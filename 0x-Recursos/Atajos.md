### [[Índice de programación]]
### [[Dashboard]]

|  #  | Combinación de teclas |                                  Uso                                   |
| :-: | :-------------------: | :--------------------------------------------------------------------: |
| 1.  |       ! + Enter       |        Añade la estructura base para empezar el código en HTML         |
| 2.  |     .element + *      |     Estiliza el primer elemento después de la clase en la formula      |
| 3.  |   Ctrl + Shift + F    |                      Organiza todos los elementos                      |
| 4.  |     Alt + ↓ or ↑      |      Permite mover una linea de bloque hacia arriba o hacia abajo      |
| 5.  |       Ctrl + D        |       Selecciona la siguiente coincidencia próxima para editarla       |
| 6.  | Shift + Alt + ↓ or ↑  | Duplica la linea o bloque para escribir en diferentes lineas de código |
